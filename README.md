## 🚀 Deployment Blueprint

### Prerequisites
- Node.js (v20+)
- Supabase Project
- Vapi.ai API Key
- OpenRouter API Key

### Installation
```bash
git clone https://github.com/AnishkumarProjects05/AI-Interview-Questioner.git
cd ai-interview
npm install
```

### Environment Configuration
Create a `.env.local` file:
```env
NEXT_PUBLIC_SUPABASE_URL=your_url
NEXT_PUBLIC_SUPABASE_KEY=your_key
OPEN_ROUTER_API_KEY=your_key
NEXT_PUBLIC_VAPI_KEY=your_key
NEXT_ROUTER_INTERVIEW_HOST_ID=https://your-domain.vercel.app/interview
```
